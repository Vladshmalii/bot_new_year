# DnD WebApp - Система управления настольной ролевой игрой

Простое веб-приложение для управления настольной ролевой игрой с разделением на интерфейс Игрока и Мастера.

## Технологический стек

- **Frontend**: React 18, TypeScript
- **Хранение данных**: JSON файл + localStorage

## Быстрый старт

### 1. Установка зависимостей

**Frontend:**
```bash
cd frontend
npm install --legacy-peer-deps
```

### 2. Запуск

**Frontend:**
```bash
cd frontend
npm run dev
```

Приложение будет доступно:
- Frontend: http://localhost:3000

Данные хранятся в `localStorage` браузера. Мастер может экспортировать/импортировать данные через интерфейс управления данными.

## Авторизация

Единое поле ввода:
- **Игрок**: Введите имя персонажа (например, "Гэндальф")
- **Мастер**: Введите пароль `2365`

Система автоматически определит тип входа по введенным данным.

## Функциональность

### Интерфейс Игрока

- Профиль персонажа
- Характеристики (HP, статы, способности)
- Инвентарь
- Записки от мастера
- Броски кубиков

### Интерфейс Мастера

- Дашборд со всеми персонажами
- Мониторинг бросков кубиков
- Управление локациями
- Отправка подсказок
- Управление инвентарём
- Создание мобов
- Импорт данных из Excel

## Управление данными

Мастер может экспортировать и импортировать данные в формате JSON через интерфейс "Управление данными".

Формат JSON файла:
- `characters` - массив персонажей
- `locations` - массив локаций
- `mobs` - массив мобов
- `items` - массив предметов
- `character_items` - связи персонажей и предметов
- `notes` - записки
- `dice_rolls` - история бросков кубиков

## Мобильная адаптивность

Приложение полностью адаптировано для мобильных устройств.

## Деплой на GitHub Pages

Для публикации приложения на GitHub Pages:

1. **Создайте репозиторий на GitHub** (если еще не создан)

2. **Обновите homepage в `frontend/package.json`**:
   ```json
   "homepage": "https://YOUR_USERNAME.github.io/bot_new_year"
   ```
   Замените `YOUR_USERNAME` на ваш GitHub username.

3. **Инициализируйте Git** (если еще не сделано):
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git remote add origin https://github.com/YOUR_USERNAME/bot_new_year.git
   git branch -M main
   git push -u origin main
   ```

4. **Включите GitHub Pages** в настройках репозитория:
   - Settings → Pages
   - Source: Branch `gh-pages`, Folder `/ (root)`

5. **Задеплойте**:
   ```bash
   cd frontend
   npm run deploy
   ```

Сайт будет доступен по адресу: `https://YOUR_USERNAME.github.io/bot_new_year`
